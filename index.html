LastFM Java API

last.fm JAVA wrapper for online track and album metadata information processing

About LastFM

last.fm believe that there is an infinite universe of music out there and they want to help the world navigate, explore, and enjoy even more of it. They bring together our favourite music services and join up listening, watching and sharing to connect our musical world.
They do it by:
1. Explore: They help us explore a vast world of music and learn about new music we might like.
2. Connect: They help us to see what other people like and share our passion for music with them.
3. Revisit: They help us track what we’ve heard and use this to power recommendations and re-discovery.

What's in this API

This is a java wrapper API for last.fm REST-ful web services.This API have all built in serialisation, deserialisation techniques to automate fetch of music metadata and Map to built-in POJO beans. This API currently supports only explore part.

Example

String apiKey = "yourapikey";
LastFMGeoTopTracks geoTopTracks = new LastFMGeoTopTracks (apiKey);
// fetch only 5 tracks
geoTopTracks.setLimit (5);
GeoTopTracks topTracks = geoTopTracks.fetchGeoTopTracks ("India");
// this will print all the top tracks currently in Indian geo area along with artist name
topTracks.track.forEach (consumer -> System.out.println(consumer.name + " by "+ consumer.artist.name));
